### Offline Doc Tools
If your internet cuts off often or is slow:
1.  **DevDocs.io**: Enables offline mode. You can download docs for HTML, CSS, JS, Python, React.
2.  **Zeal**: A desktop offline documentation browser for Windows/Linux.

### Lightweight Editors
Don't use heavy IDEs (like Visual Studio Enterprise or Android Studio) if you have an old laptop.
*   **Notepad++**: Extremely light.
*   **Sublime Text**: Very fast, low memory usage.
*   **Vim/Neovim**: Runs in terminal, zero lag (High learning curve, but worth it).
