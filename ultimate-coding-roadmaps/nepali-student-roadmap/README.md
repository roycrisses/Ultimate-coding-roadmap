## ğŸ›¤ï¸ Customized Learning Path for Nepal

### Step 1: Foundations (+2 Level / 1st Year)
Don't worry about AI/ML yet. Focus on the basics.
- **English:** IT ma English important cha. Basic documentation padhna sakne huna parcha.
- **Computer Basics:** Typing speed badhauhus (Try `monkeytype.com`).
- **Logic:** C or Python bata start garnuhus. (College ma C padhaucha bhane tei ramrari siknus).

### Step 2: The Core (Bachelor 2nd Year)
Select *one* path. "Sabai kura" sikchu bhanera confusion huncha.
- **Web Development (Best for Freelancing in Nepal)** -> HTML/CSS -> JS -> React + Node.
- **Mobile Apps** -> Flutter (Nepal ma Flutter ko demand high cha).
- **Backend/Enterprise** -> Java or .NET (Banks and corporate jobs in KTM use this).

### Step 3: Internship Ready (3rd Year onwards)
- **Projects:** Make at least 3 solid projects.
- **CV:** Make a clean CV (No photo needed generally for intl remote, but Nepal ma photo rakhda huncha).
- **Networking:** Join "Developers Nepal" Facebook group or LinkedIn.
